## 📝 Описание проекта

Простое веб-приложение на Django, которое позволяет пользователям вводить свои имена и получать персонализированное приветствие. Приложение сохраняет имена в базу данных и отображает список последних посетителей.

## ✨ Функциональность

- ✅ Форма для ввода имени пользователя
- ✅ Персонализированное приветствие после отправки
- ✅ Сохранение имен в базу данных SQLite
- ✅ Отображение последних 10 посетителей
- ✅ Обработка ошибок (пустое поле ввода)
- ✅ Защита от CSRF-атак
- ✅ Современный адаптивный дизайн

## 🎨 Особенности интерфейса

- Градиентный фон (фиолетово-синий)
- Плавные анимации появления элементов
- Карточка с формой и тенью
- Цветовая индикация сообщений:
  - 🟢 Зеленый - успешное приветствие
  - 🔴 Красный - ошибка ввода
- Автоматическое скрытие сообщений через 5 секунд
- Список последних посетителей в виде цветных пузырьков

## 🛠 Технологии

- **Backend**: Django 4.x
- **Database**: SQLite
- **Frontend**: HTML5, CSS3, JavaScript
- **Безопасность**: CSRF-защита Django
